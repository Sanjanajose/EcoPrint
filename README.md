# EcoPrint – Enterprise Print Management Platform

EcoPrint is an enterprise print management platform designed to support secure authentication, print workflows, monitoring, queue management, and administrative operations.

## Tech Stack

* Java
* Spring Boot
* REST APIs
* JWT Authentication
* MySQL
* Maven
* Role-Based Access Control (RBAC)

## Features

* User Authentication & Authorization
* Role & Permission Management
* Print Job Submission & Queue Management
* Job Status Monitoring
* Activity Logging & Tracking
* Administrative Controls
* Secure REST API Development

## Project Highlights

* Built using Java and Spring Boot architecture
* Implemented authentication and authorization workflows
* Developed scalable backend APIs for business operations
* Designed enterprise-focused workflow management features
* Applied secure development and modular backend practices

## Purpose

This project demonstrates backend engineering, enterprise application development, API design, authentication, and scalable system design concepts.
